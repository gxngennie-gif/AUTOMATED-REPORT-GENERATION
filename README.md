#DESCRIPTION OF THE TASK:
*Project Title*:
Employee Salary Analysis and PDF Report Generation Using Python

 *Project Description*:
This project was developed using Python in Visual Studio Code.
The main objective of this project is to analyze employee salary data from a CSV file and automatically generate a professional PDF report.
The system reads employee data (such as Name, Department, and Salary), performs salary analysis, and creates a well-structured PDF report that includes:
Total number of employees:
Average salary
Highest salary
Lowest salary
Department-wise average salary table
This project demonstrates data analysis, automation, and report generation using Python libraries.

 *Tools and Technologies Used*
#Programming Language
Python 3.13
#Code Editor
Visual Studio Code
#Libraries Used
pandas
Used for reading CSV files
Performing salary calculations
Grouping data department-wise
eportlab
Used for generating PDF files
Creating formatted tables
Adding headings and styling

 *How the Project Works*

1️. The program reads employee data from data.csv.
2️. It calculates salary statistics using pandas.
3️. It groups employees by department and finds average salary.
4️. It creates a structured PDF report using ReportLab.
5️. The final output is saved as Employee_Report.pdf.

*Output*
A professional PDF file containing:
Salary summary
Department-wise salary table
Proper formatting and structure
